            <h1 class="firstHeading">(10094) Eijikato</h1>
            <div id="bodyContent">
                <h3 id="siteSub">De Wikipedia, la enciclopedia libre</h3>
                <div id="contentSub"></div>
                <!-- start content -->
                <p><b>Eijikato</b> (asteroide 10094) es un <a href="../../../../articles/a/s/t/Asteroide.html" title="Asteroide">asteroide</a> del <a href="../../../../articles/c/i/n/Cintur%C3%B3n_de_asteroides.html" title="Cinturón de asteroides">cinturón principal</a>, situado a 2,2055512 <a href="../../../../articles/u/n/i/Unidad_astron%C3%B3mica.html" title="Unidad astronómica">UA</a>. Tiene una <a href="../../../../articles/e/x/c/Excentricidad.html" title="Excentricidad">excentricidad</a> de 0,144702 y un <a href="../../../../articles/p/e/r/Periodo_orbital.html" class="mw-redirect" title="Periodo orbital">periodo orbital</a> de 1.512,5 días (4,14 años).</p>
                <p>Su <a href="../../../../articles/v/e/l/Velocidad_orbital.html" title="Velocidad orbital">velocidad orbital</a> media es de 18,54782159 km/s y tiene una <a href="../../../../articles/i/n/c/Inclinaci%C3%B3n_orbital.html" title="Inclinación orbital">inclinación</a> de 15,66164º. Fue descubierto el <a href="../../../../articles/2/0/_/20_de_febrero.html" title="20 de febrero">20 de febrero</a> de <a href="../../../../articles/1/9/9/1991.html" title="1991">1991</a> por <a href="../../../../articles/t/s/u/Tsutomu_Seki_9595.html" title="Tsutomu Seki">Tsutomu Seki</a>.</p>
                <p><br /></p>
